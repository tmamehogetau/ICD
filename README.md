# 居酒屋開発会議

オンライン対応のカード調整会議ゲームです。3〜6人が招待リンクで同じ会議室へ入り、各自の端末から同時に制作と秘密投票を行います。

## 起動

```powershell
npm.cmd start
```

`http://localhost:4173` を開き、「会議室を作成する」から始めます。表示された招待リンクを参加者へ共有してください。

## オンライン版の扱い

- 手札・未公開票はサーバーが保持し、本人の端末にだけ返します。
- 同じ端末では、会議コードと復帰情報をブラウザに保存するため、再読み込み後も復帰できます。
- 会議の途中状態は `data/online-rooms.json` に自動保存され、サーバーを再起動しても復元されます。
- 公開する際は、書き込み可能な永続ディスクを持つ単一の Node.js サーバーに配置し、HTTPS の URL を招待リンクとして使ってください。複数台構成にする場合は、ルーム状態を共有DBへ移す必要があります。
